# Subarrays with OR = 1

## Problem Description

You are given a **binary array** `A` of length `N`, where each element is either `0` or `1`.

Your task is to **count the number of subarrays** such that the **bitwise OR** of all elements in the subarray is **1**.

---

## Constraints

- `1 ≤ N ≤ 10^4`
- `A[i] ∈ {0, 1}`
