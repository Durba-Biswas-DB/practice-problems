# Set Bit

## Problem Description

You are given two integers **A** and **B**.

You need to **set the A-th bit and the B-th bit in the number 0**, and return the resulting value in **decimal (base-10)** number system.

### Notes
- Bit positions are **0-indexed**
- The **Least Significant Bit (LSB)** has index `0`

---

## Problem Constraints

- `0 <= A <= 30`
- `0 <= B <= 30`
