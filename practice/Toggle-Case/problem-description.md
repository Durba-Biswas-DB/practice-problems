# Toggle Case of a String

## Problem Description

You are given a character string **A** having length **N**, consisting only of lowercase and uppercase Latin letters.

Your task is to **toggle the case of each character** in the string:

- Uppercase characters become lowercase  
  - Example: `'A' → 'a'`, `'Z' → 'z'`
- Lowercase characters become uppercase  
  - Example: `'e' → 'E'`, `'m' → 'M'`

Return the modified string after toggling the case of every character.

---

## Problem Constraints

- **1 ≤ N ≤ 10⁵**
- **A[i] ∈ ['a'–'z', 'A'–'Z']**
