# Single Number

## Problem Description

Given an array of integers **A**, every element appears **twice** except for **one** element.  
Your task is to find and return the element that occurs **only once**.

### Notes
- The solution must run in **linear time**.
- Do **not** use extra memory.

## Problem Constraints

- `1 <= |A| <= 2,000,000`
- `0 <= A[i] <= INT_MAX`
