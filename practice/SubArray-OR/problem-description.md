# Subarray OR

## Problem Description

You are given an array of integers `A` of size `N`.

The value of a subarray is defined as the **bitwise OR** of all elements in that subarray.

Return the **sum of values of all subarrays of `A` modulo (10^9 + 7)`**.

---

## Constraints

- 1 <= N <= 10^5
- 1 <= A[i] <= 10^8
