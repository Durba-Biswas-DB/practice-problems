# Check Pair Sum

## Problem Description

Given an Array of integers **B**, and a target sum **A**.  
Check if there exists a pair **(i, j)** such that:

- `B[i] + B[j] = A`
- `i != j`

Return `1` if such a pair exists, otherwise return `0`.

---

## Problem Constraints

- `1 <= Length of array B <= 10^5`
- `0 <= B[i] <= 10^9`
- `0 <= A <= 10^9`
