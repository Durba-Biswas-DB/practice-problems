# Appear Once

## Problem Description

You are given an array of positive integers **A**.  
In this array, **exactly two integers appear only once**, while **all other integers appear exactly twice**.

Your task is to **find the two integers that appear only once**.

### Note
- Return the two numbers in **ascending order**.

---

## Problem Constraints

- `2 <= |A| <= 100000`
- `1 <= A[i] <= 10^9`
