# Appear Thrice

## Problem Description

Given an array of integers, every element appears **thrice** except for one element, which appears **exactly once**.

Your task is to find the element that does **not** appear three times.

---

## Constraints

- `2 <= |A| <= 5 * 10^6`
- `0 <= A[i] <= INT_MAX`
- **Linear time complexity required**
- **No extra memory allowed**
